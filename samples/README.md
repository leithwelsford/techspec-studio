# Sample Files

This directory contains sample and test files for TechSpec Studio.

## Files

### sample-brs.md
**Purpose**: Example Business Requirement Specification (BRS) document for testing.

**Use Case**: Test the BRS-to-TechSpec generation workflow.

**Content**: Realistic 5G telecommunications BRS (318 lines) with:
- Customer requirements
- Architecture descriptions
- Call flow scenarios
- Non-functional requirements

**How to Use**:
1. Open TechSpec Studio
2. Navigate to "BRS" tab
3. Upload `sample-brs.md`
4. Click "Generate Specification from BRS"
